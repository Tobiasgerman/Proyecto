¿En qué consiste el Multi Wordle?

El Multi Wordle es un sitio en el que los diferentes modos de categorías y tipos de wordle se mezclan.

Paleta de colores:

A

Si el color es verde significará que el dato/letra que haya ingresado el usuario es correcto y está bien posicionado.

A

Si el color es amarillo significará que el dato/letra que haya ingresado el usuario es correcto pero está mal posicionado.

A

Si el color es gris significará que el dato/letra que haya ingresado el usuario es incorrecto y está mal posicionado.
