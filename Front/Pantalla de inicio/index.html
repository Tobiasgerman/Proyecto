Â¿En que consiste el Multi Wordle?

El Multi Wordle es un sitio en el que los diferentes modos de categorias y tipos de wordle se mezclan.

Paleta de colores:

A

Si el color es verde significara que el dato/letra que haya ingressado el usuario es correcto y esta bien posicionado.

A

Si el color es amarillo significara que el dato/letra que haya ingresado el usuario es correcto pero esta mal posicionado.

A

Si el color es gris significara que el dato/letra que haya ingresado el usuario es incorrecto y esta mal posicionado.
